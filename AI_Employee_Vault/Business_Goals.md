## Operational Rules

### Task Processing

- All tasks: Inbox → Needs_Action → Approval → Done
- Critical tasks: Process within 24 hours
- Standard tasks: Process within 48 hours
- Tasks > 5 days old: Escalate automatically

### Approval Thresholds

| Action Type | Auto-Approve | Human Approval Required |
|-------------|--------------|-------------------------|
| Task status updates | Always | Never |
| File operations | Always | Never |
| Email responses | To known contacts | New contacts |
| Social media posts | Never | Always |
| Payments < $100 | Recurring only | One-time |
| Payments > $100 | Never | Always |

### Communication Standards

- Client emails: Respond within 4 hours
- Internal tasks: Update daily
- External communications: Approval required
- Status reports: Weekly on Fridays
