# Skill: LinkedIn Poster

## Description
Automates LinkedIn posting using Playwright browser automation.
Creates business posts, schedules content, and tracks engagement.

## Trigger
- Scheduled posting times
- Manual trigger for immediate posts
- Command: `/post-linkedin`, `/schedule-linkedin`

## Inputs
| Input | Type | Required | Description |
|-------|------|----------|-------------|
| content | String | Yes | Post content (max 3000 chars) |
| post_type | String | No | text, image, article (default: text) |
| image_path | String | No | Path to image (if post_type is image) |
| schedule_time | DateTime | No | When to post (if scheduling) |
| hashtags | Array | No | Hashtags to include |

## Process

### Step 1: Content Preparation
1. Validate content length (max 3000 characters)
2. Format hashtags
3. Check for prohibited content
4. Generate preview

### Step 2: Request Approval (HITL)
1. Create approval request in `/Pending_Approval/`
2. Include post preview
3. Wait for human approval

### Step 3: Browser Automation (After Approval)
1. Launch Playwright with persistent session
2. Navigate to LinkedIn
3. Verify login status
4. Create new post

### Step 4: Post Execution
1. Enter post content
2. Add image if provided
3. Click post button
4. Verify post success

### Step 5: Log & Notify
1. Save post record to `/Done/`
2. Log to daily log
3. Update Dashboard.md

## Output
| Output | Type | Description |
|--------|------|-------------|
| post_id | String | LinkedIn post identifier |
| status | String | posted, scheduled, failed |
| posted_at | DateTime | When post was published |
| url | String | Link to the post |

## Content Guidelines

### Recommended
- Business updates and achievements
- Industry insights and tips
- Professional announcements
- Thought leadership content

### Prohibited
- Spam or promotional overload
- Controversial political content
- Unverified claims
- Competitor attacks

## Example Usage

```
Skill: linkedin_poster
Input:
  content: "Excited to announce our new AI automation project! 🚀
            Building the future of autonomous business operations.
            #AI #Automation #Innovation"
  post_type: "text"
  hashtags: ["AI", "Automation", "Innovation"]

Output:
  status: "pending_approval"
  approval_file: "/Pending_Approval/LINKEDIN_20260220_143500.md"
```

## Browser Session Management

1. Use persistent Playwright context
2. Store session in secure location (not in vault)
3. Handle 2FA if required
4. Implement session refresh logic

## Safety Rules
- All posts require human approval
- Never post without explicit confirmation
- Rate limit: max 3 posts per day
- Never access DMs or connections without approval
